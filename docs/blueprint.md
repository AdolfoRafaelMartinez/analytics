# **App Name**: Block Number Now

## Core Features:

- QuickNode Endpoint Connection: Establish a connection to the QuickNode endpoint using the provided API key to access blockchain data.
- JSON RPC Method Call: Construct and execute an 'eth_blockNumber' JSON RPC method call.
- Result Display: Display the resulting value (block number) from the JSON RPC call in a user-friendly format.
- Error handling: Provide error messages for API errors, authentication issues, and other connectivity problems

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to represent blockchain and technology.
- Background color: Very light blue (#E8EAF6).
- Accent color: A muted teal (#4CAF50), analogous to the blue primary, to signal valid data retrieval.
- Body and headline font: 'Inter', a sans-serif font, for a clean and modern user experience.
- Simple, line-based icons to represent blocks and data flow.
- Clean, single-column layout to prominently display the block number and related information.
- Subtle transitions or loading animations when fetching and updating the block number.
